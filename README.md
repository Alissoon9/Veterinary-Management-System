# Veterinary-Management-System
Sistema de gestión para veterinaria desarrollado con Flask, PostgreSQL, HTML y CSS.

Sistema web desarrollado con Python, Flask y PostgreSQL para administrar la información de una clínica veterinaria.

Funcionalidades:
Registro de Dueños
Registro de Mascotas
Registro de Veterinarios
Registro de Citas
Visualización de datos
Consultas JOIN
Interfaz desarrollada con HTML y CSS
Tecnologías utilizadas:
Python
Flask
PostgreSQL
HTML
CSS
